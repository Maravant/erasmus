Se recomienda seguir el siguiente formato para crear un icono:

<i class="icon icon--colegios" aria-hidden="true"></i>

Reemplazar icon--colegios por cualquiera de las clases anteriores.

También se puede agregar la clase icon--small para imprimir un icono más pequeño de 32px.

Botones

Simples

Botones simples con clase button. Acepta enlaces, botones e inputs del tipo submit.
Simple 
<a href="#" class="button">Simple</a>
<button class="button">Enviar</button>
<input type="submit" class="button" value="Aceptar">
<button class="button" disabled>Deshabilitado</button>

Claros

Botones en fondo oscuro con clase button--dark y button--transparent.
Simple 
<a href="#" class="button button--dark">Simple</a>
<button class="button button--transparent">Aceptar</button>

Tamaños

Diferentes tamaños de botones button--big y button--small.
Simple Enviar 
<a href="#" class="button button--big">Simple</a>
<a href="#" class="button button--small">Enviar</a>

Formas

Botones redondos con button--circle. 
<a href="#" class="button button--circle button--big">
  <i class="icon icon--small icon--colegios" aria-hidden="true"></i>
</a>

<a href="#" class="button button--circle">
  <i class="icon icon--small icon--pointer" aria-hidden="true"></i>
</a>

Botones Radio

Sólo es seleccionable uno a la vez con el mismo nombre. Se crea con la etiqueta input type="radio" wrapeada en un label con clase input-radio. 
<label>

    <input type="radio" name="ra" value="1" checked><span>Sí</span>
  </label>
  <label class="input-radio">
    <input type="radio" name="ra" value="0"><span>No</span>
  </label>
</label>
